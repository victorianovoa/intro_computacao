História das Linguagens de Programação

Século XIX

Ada Lovelace escreve o primeiro algoritmo da história.

Década de 1950

FORTRAN (1957), LISP (1958), COBOL (1959) surgem para resolver cálculos científicos e dados comerciais.

Década de 1970

C (1972) nasce para o sistema Unix. Smalltalk e a Orientação a Objetos aparecem.

Anos 1990

Java, Python, JavaScript revolucionam a web e a multiplataforma.

Hoje e Futuro

Rust, Go, Kotlin e linguagens para IA dominam as novas tendências.
